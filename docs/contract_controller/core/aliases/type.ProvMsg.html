Any(AnyMsg)

CosmosMsg::Any is the replaces the “stargate message” – a message wrapped in a protobuf Any that is suppored by the chain. It behaves the same as CosmosMsg::Stargate but has a better name and slightly improved syntax.

This is feature-gated at compile time with cosmwasm_2_0 because a chain running CosmWasm < 2.0 cannot process this.
